Hydroboy

Prototyp für den Aufbau unserer hydroponischen Anlage ist eine Säule, die wir der Hydroboy genannt haben. Beim Entwurf dieses Aufbaus sollten verschiedene Kriterien erfüllt werden. Im Vordergrund steht der Wissenstransfer, wie hydroponische Anlagen funktionieren und auf individuelle Anforderungen / Entwürfe abgestimmt werden können.

Was macht den Hydroboy aus?

  • Einfaches Handling
  • Bauteile sollten im lokalen Baumarkt vorhanden sein
  • Schneller Aufbau / Nachbau möglich
  • Erweiterbar / skalierbar
  • Raum und platzsparend
  • Möglichst nur Open Source Komponenten
Der Hydroboy, unsere Anleitung für das Mischen und Verwendung von Dünger sowie der quelloffenen Code bieten eine Basis auf der eigene Ideen entwickelt, umgesetzt und geteilt werden können.
